# Bright_Coffee-Shop-Sales-Data-Analysis

This project focuses on analyzing transactional sales data from Bright Coffee Shop to generate actionable business insights for a newly appointed CEO. The goal is to support data-driven decision-making aimed at increasing revenue and improving product performance.

As a Junior Data Analyst, this project demonstrates end-to-end data handling — from data ingestion and transformation (ETL) to analysis and visualization, culminating in strategic business recommendations


#The primary objectives of this analysis are:

1.Identify top-performing products by revenue
2.Determine peak sales periods during the day
3.Analyze sales trends across products and time intervals
4.Provide actionable recommendations to improve business performance
